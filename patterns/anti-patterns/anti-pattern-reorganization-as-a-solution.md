# Anti-Pattern: Reorganization as a Solution

#### Description

Structural changes are used to address flow and dependency problems without addressing underlying system constraints.

#### Why It Emerges

* Org charts are visible levers
* Reorgs signal action
* System issues feel intangible

#### Consequences

* Disruption without improvement
* Increased coordination cost
* Fatigue and cynicism

#### Signal

> “We just need to realign the teams.”
