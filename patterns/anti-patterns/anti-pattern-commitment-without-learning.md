# Anti-Pattern: Commitment Without Learning

#### Description

Commitments are made before feasibility, risk, or uncertainty is reduced.

#### Why It Emerges

* External pressure
* Desire to appear decisive
* Learning seen as delay

#### Consequences

* Firefighting
* Quality erosion
* Trust loss
* Escalation culture

#### Signal

> “We’ll figure it out as we go — but the date is fixed.”
